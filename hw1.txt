 1. Как скопировать на свой компьютер репозиторий из github.com, что 
должно быть сделано предварительно? --
Установить Git (проверить наличие git -v)
настроить Git git config --global user.name git config --global user.email
клонировать ссылку на репозиторий из GitHub
в консоли ввести git clone "url"

2. Как получить на локальный компьютер изменения с github.com
git pull

Какую информацию нам дают команда :
git status - состояние файлов (изменения, которые не были "закоммиченны")
git log - история "коммитов"
